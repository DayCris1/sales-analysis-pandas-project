# 📊 Análise de Dados de Vendas (Projeto Simulado)

Este projeto simula uma demanda real de trabalho, onde um gestor solicita uma análise inicial de dados de vendas.
O objetivo é praticar conceitos básicos de análise de dados utilizando Python e Pandas.

## 🛠️ Etapas da Análise

- Leitura e exploração inicial dos dados
- Análise estatística descritiva
- Filtragem de dados
- Identificação de produtos mais vendidos
- Agrupamento por categoria
- Cálculo de faturamento

---

## 📈 Principais Insights

Realizei uma análise inicial do dataset de vendas enviado.

Resumo dos dados:

* O dataset possui 10 registros e 5 colunas.
* Não foram identificados valores faltantes.

Principais insights:

* O produto mais vendido foi o *Mouse*, com 3 ocorrências.
* Os produtos mais frequentes são: *Mouse*, *Teclado* e *Cadeira*.
* A categoria **Móveis** possui maior preço médio (≈ 366,67) em comparação com Eletrônicos (≈ 278,57).
* Os produtos com maior valor unitário são os monitores (R$ 800).

Faturamento:

* O produto com maior faturamento total foi o *Monitor* (R$ 1.600).
* Em seguida: *Cadeira* (R$ 900) e *Mesa* (R$ 500).

Observações:

* A maior parte das vendas possui quantidade igual a 1 unidade.
* Há concentração de vendas em poucos produtos.

## 📌 Conclusão

A análise permitiu identificar padrões de vendas, produtos com maior desempenho e diferenças entre categorias.

Esse tipo de análise é essencial para apoiar decisões estratégicas em negócios.
